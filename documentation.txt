React without CRA
Using Babel
Babel basically acts as a transpiler.


1. npm install react
2. npm install react-dom
3. module bundler, npm install webpack --save-dev
4. text-pack for development, npm install webpack-cli --save-dev
commands to initialize webpack,monitor, etc.

5. webpack-dev server , npm install webpack-dev-server --save-dev
live reload

6. babel functionality, npm install @babel/core --save-dev

7.npm install babel-loader --save-dev
process different file types.

8. npm install @babel/preset-react --save-dev
presets for react plugins,language 

9. npm install html-webpack-plugin --save-dev
html file creation to ease
10. @babel/preset-env



Webpack application configuration
1. entry: react basic starting code
2. bundled file output-output
3. htmlWebPack-plugin, make index.html 
4. module rules, i.e all js files need to be loaded by babel-loader, with required present.
